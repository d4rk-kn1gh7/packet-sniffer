# Packet Sniffer
A simple packet sniffing tool written in python, which captures data on a raw socket, and has a web server to display data. A client-server model is used for interaction between the tool and the web server. 

This project was made as our S4 Networks Project

## Dependencies
+ argparse
+ pwntools
+ requests
+ flask
+ flask_pymongo
+ tqdm
